The game snake is based on basic javaScript and html canvas. The game area is a 320px x 480px canvas with the snake head as a 16x16 box. The head moves ahead at a predefined speed that is called pace. It is named as myGamePiece.pace where myGamePiece is the head of the snake.

The score variable is an integer that keeps count of the score. The variable btncount is an integer that keeps track of number of times direction of snake movement is changed.

The function setPace changes the value of myGamePiece.pace to the variable input_pace that is taken as input from the html.

The apple is a 16x16 px object on the canvas, collision with which increases the score and resets the apple. The apple is created using the function apple.set. This function sets apple.x and apple.y randomely anywhere in the canvas.

The maze is a set of obstacles that will terminate game on collision

As the page loads, the function myGameArea.start is called. This function creates the canvas, sets the score to zero, creates the apple and starts a timed interval that calls updateGameArea function periodically.

updateGameArea() function calls myGamePiece.newPos() and myGamePiece.update(). The newPos function checks whether the user has changed direction, creates a new snake head, checks whether the snake head is colliding with the maze, itself or the edge, checks whether the head is colliding with the apple and finally calls clearFeces. The final function clears the previous head such that the snake length is maintained.

The tail is a Queue data structure that stores the previous coordinates of the previos heads of the snake. Data is enqueued in myGamePiece.newPos and dequeued in clearFeces.

myGamePiece.update() function applies color to the new snake head and the new apple.

The terminate() function is called whenever the snake head collides with the edge, the maze or it's own body. This function clears the canvas, alerts the score and calls myGameArea.start() once again to restart the game.

The changeTheme function is used to change the background color as the score increases.

moveleft(), moveright(), moveup(), movedown() change direction of motion by changing value of myGamePiece.speedX or myGamePiece.speedY
